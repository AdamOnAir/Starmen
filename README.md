A hexeditor made having Earthbound in mind.








## Compile and Run

Compile with GCC:
```
git clone https://gitHub.com/FBDev64/Starmen.git
cd Starmen
gcc -o starmen main.c
```

Compile with CMake:
```
git clone https://gitHub.com/FBDev64/Starmen.git
cd Starmen
mkdir build && cd build
cmake ..
make
```
